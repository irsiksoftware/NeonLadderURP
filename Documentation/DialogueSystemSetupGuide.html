üéÆ Unity Editor Workflow

1Scene Setup Workflow

  1. Create empty scene or use existing level
  2. Create DialogueSceneConfiguration asset
  3. Drag character prefabs to Left/Right Character slots
  4. Set positions: Left (-2,0,0), Right (2,0,0)
  5. Configure trigger box size (default: 2x2x2)
  6. Set conversation name (must match Dialogue Database)
  7. Click "Generate Dialogue Scene"

2Pixel Crushers Integration

  1. Open Tools ‚Üí Pixel Crushers ‚Üí Dialogue System ‚Üí Dialogue Editor
  2. Create new Database or open existing
  3. Add Actors for your characters (match ActorName in config)
  4. Create Conversations (match ConversationName in config)
  5. Set up Text Tables for localization
  6. Assign Database to DialogueSystemController
